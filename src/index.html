About the app

Hi my name is Julio Jaramillo the autor of this mini app. Im a FE developer since almost 8 years and I want to show a demostration how to make a reusable geometrical visual app in Javascript.

¿What is this app about?

This is an canvas application that will demostrate how the Parallelogram works, and how an ellipse can be inscribed on it.

¿How it works?

The user have to create 3 points in the canvas draw screen, to create a new point the user should double click on the inside the canvas and you will see a red dot is being created. You should create 3 points along the canvas screen, once you've created them you will see a new 4 point created automatically filled in grey, this new grey point is going to change coordinates position to make sure the shape always be a parallelogram. Now we have 4 points in the screen a Parallelogram is going to be created within a blue line stroke. Also there will be a new ellipse (within yello line stroke) created that's going to be inscribed inside the parallelogram area. You can move the created points along the screen, and you will see the parallelogram fit to users movements.

Extra features

The user would have the posibility to toggle visibility of the Parallelogram lines and the ellipsed inscribed, also you can lock/unlock the parallelogram fit, that means you can move the points around the canvas screen and the parallelogram can be deformed
